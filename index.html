Proceso de producción

Diagrama de flujo

La creación del Diagrama de Flujo fue un paso esencial para la validación de la arquitectura del sistema. Este proceso resultó fundamental para estructurar la lógica interna y garantizar la coherencia total del flujo de datos antes de iniciar la fase de codificación.

Ver en Figma →

Diseño de pantallas

La fase de prototipado fue indispensable para sentar las bases de la interfaz y validar su usabilidad. Mediante este proceso, pudimos estructurar la navegación de manera eficiente y confirmar que se cumplieran los estándares de diseño y la consistencia visual requerida, facilitando una transición fluida hacia el código.

Ver en Figma →

Implementación

Para la fase de desarrollo, la elección de p5.js y Processing fue fundamental para la visualización interactiva. Trabajamos con arrays como pilar de nuestra estructura de datos, lo cual nos permitió modelar con precisión las relaciones y estados de los componentes del proyecto. Esta arquitectura de codificación garantizó que el sistema mantuviera una gran capacidad de escalabilidad y un rendimiento fluido.
